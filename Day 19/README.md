# 🐍 100 Days of Code – Python Bootcamp (Projects: Day 19)

This repository contains two fun and interactive Python projects using the `turtle` graphics module:

- 🖍️ **Etch-A-Sketch Drawing App** (Day 19)
- 🐢 **Turtle Race Game** (Day 19)

These projects are based on the [100 Days of Code: Python Bootcamp by App Brewery](https://www.udemy.com/course/100-days-of-code/).

---

## 🖍️ Day 19: Etch-A-Sketch Drawing App

Recreate the classic Etch-A-Sketch toy using Python! Control a turtle to draw on the screen with your keyboard.

### 🎮 Controls
- `W` – Move Forward  
- `S` – Move Backward  
- `A` – Turn Left  
- `D` – Turn Right  
- `C` – Clear the Screen and Reset Turtle Position

### 💡 Features
- Real-time turtle drawing
- Keyboard event binding
- Clear/reset functionality

## 🐢 Day 20: Turtle Race Game

A fun turtle race simulation where you place a bet on the color of the turtle you think will win!

## 🎮 How It Works

- Input your color bet via pop-up
- Turtles move with random speeds
- If your turtle wins, you win!

## 💡 Features

- `random` module for varied speeds
- Input-based interaction
- Win/lose result display
